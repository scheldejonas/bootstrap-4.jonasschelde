Om Full Stack Konferencen

Den smukke by Helsingør vil skabe rammerne for denne Full Stack Konference!

Oplev fremtidens JavaScript med industriens professionelle udviklere og idealister. Oplev nye teknologier og opgrader din karriere som web udvikler.

Eksperterne taler

Vores præsentation af talere er lige blevet offentligt, så vent ikke for længe med at købe dine billet.!

Vil du gerne møde det internationale JavaScript Fællesskabet og dele dine evner med nogle af verdens top eksperter, hackere og skabere? Vær den første til at vide hvad der kan forventes i denne verden af JavaScript.

Full Stack Konference er fast besluttet på at være åben for alle. Vi ser frem til en spændende og lærig dag at dele med jer.

Hvad vil du lære?

MongoDB: NoSQL database Express: Framework til Node React: JavaScript bibliotek Node.js: JavaScript miljø ES2015: Seneste version af JavaScript Babel: JavaScript compiler

Talere

Susi Vue

Susi er web udvikler og underviser, som er passioneret omkring at lave skalerbare, data drevne web applikationer. Specielt de som addresser gamle problemer med nye teknologier!

Anders Node

Anders er software ingeniør og nærmest filosofisk omkring at forlade denne jord bedre end han fandt den. Han koder til frivilige projekter, webshops og større web applikationer.

Søren Redux

Søren er en JavaScript udvikler arbejde med større web applikationer. han er også underviser som arbejder stærkt med at fjerne læringsbegrænsninger for sine studerende.

Ecma Pedersen

Ecma har fundet sin passion for programmering og undervist i over 15 år. Hun er meget begejstret for at introducere skønheden i et nyere og simplere JavaScript.

Jan Query

Jan er udvikler, forfatter til: "CSS, den manglende manual" & samtidig underviser i web udvikling.

Jonas Babel

Hele hans professionelle karrierer har han arbejdet med computere online. Han er en polyglot programmør, som elsker at bruge de rigtige værktøjer til det rette job.
